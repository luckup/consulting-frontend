# Service card images — alternate styles (website-focused)

Use these **instead of** the warm/light lifestyle prompts in `service-image-prompts.md` when you want more variety.  
**Focus:** the **website UI** for each service type. **People are optional** — prompts below default to **no humans**.

**Aspect ratio:** 16:9 · **Export:** `src/assets/pages/services/{filename}-cold.png` or `{filename}-advanced.png`

---

## Global negative prompt (all styles)

```
people, faces, hands, human silhouettes, warm golden hour, sunny yellow tone, illustration, cartoon, 3D clay, isometric infographic, abstract blobs, hologram, neon cyberpunk, messy UI, illegible lorem ipsum, watermark, logo soup, duplicate screens, low resolution
```

*(Remove `people, faces, hands` from negative prompt if you **want** a person in that shot.)*

---

## Style A — Cold colored

**Mood:** ice blue, slate, cool gray, soft cyan highlights, crisp and calm.  
**Append to every Cold prompt:**

```
cool color grading, ice blue and slate gray palette, soft cyan highlights, crisp cold atmospheric light, professional tech studio backdrop, website UI sharp and readable as hero subject, photorealistic device mockup or browser frame, no warm tones, 16:9
```

---

## Style B — Advanced

**Mood:** premium, refined, “Apple keynote” product shot — glass depth, precision layout, muted luxury.  
**Append to every Advanced prompt:**

```
ultra-premium advanced web design presentation, refined minimalist UI with subtle glassmorphism soft shadows, sophisticated neutral grays with muted orange accent CTAs, high-end device or browser mockup, studio product photography, interface fills most of frame, futuristic yet realistic professional aesthetic, 16:9
```

## Quick pick guide

| If you want… | Use |
|--------------|-----|
| Distinct from warm lifestyle photos | **Cold** or **Advanced** sets above |
| Blue / corporate tech feel | **Cold** |
| Premium keynote / Dribbble showcase | **Advanced** |
| Website is the star | Both (no human in scene) |
| One person optional | Delete `no people` and add e.g. `hand only using trackpad, no face` |
